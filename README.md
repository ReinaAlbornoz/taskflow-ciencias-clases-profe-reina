# 🌸 TaskFlow Ciencias - Gestión de Pacientes
### Proyecto desarrollado por: Reina Albornoz

Este proyecto es un servidor backend profesional desarrollado con **Node.js** y **Express** para la gestión de datos médicos de pacientes.

## 🚀 Funcionalidades
* **Servidor Express:** Configurado en el puerto `3000`.
* **Middlewares:** Registro de accesos (Logging) con fecha y hora.
* **Arquitectura:** Estructura organizada en carpetas (Routes, Controllers, Middlewares, Data).
* **Persistencia:** Manejo de datos mediante archivos JSON.

## 🛠️ Instalación y Uso
1. Clonar el repositorio:
   `https://github.com/ReinaAlbornoz/taskflow-ciencias-clases-profe-reina.git`
2. Instalar dependencias:
   `npm install`
3. Iniciar el servidor:
   `node index.js`
